My Neovim Config

A fast, minimal, and highly-customizable Neovim setup (built with Lua & lazy.nvim).

Features Installation Key Plugins Config Screenshots Contact

Features

  • Plugin management with lazy.nvim for fast startup and lazy-loading.
  • Modern editing UX: LSP, autocompletion (nvim-cmp), snippets (LuaSnip).
  • Tree-sitter powered syntax highlighting.
  • Fuzzy finding and file navigation (Telescope).
  • Small, ergonomic keymaps and sensible defaults.

Installation

  1. (Optional) Backup your existing config: 
    mv ~/.config/nvim
						~/.config/nvim.bak
  2. Clone the repo: 
    git clone
						https://github.com/DavidBalishyan/nvim.git ~/.config/nvim
  3. Start Neovim. On first run lazy.nvim will automatically install plugins: 
    nvim

Some LSPs or external tools may need to be installed separately (e.g. rust-analyzer, gopls, pyright).

Config snippets

init.lua

-- main entry: ~/.config/nvim/init.lua
				require("lazy").setup("lua/plugins")
				require("options")
				require("keymaps")

example keymap

				vim.keymap.set('n', 'w', ':w', { noremap = true })

example plugin spec

{
				"nvim-treesitter/nvim-treesitter",
				build = ":TSUpdate",
				lazy = false,
				}
